ОТВЕТЫ НА ВОПРОСЫ (Сергеев И)

1. MetaMask, Coinbase wallet, Trust Wallet, Hyperledger, Ambire, Exodus, e.t.c.

2. Позволяет используя web3 подключиться к ноде в какой-либо сети, за счет чего можно, например,
разместить в ней смарт контракт или произвести транзакцию.

3. Данные преобразуются в JSON формат для передачи по протоколу JSON-RPC. Т.е в json описании
запроса будет содержатся имя вызываемого в смарт конракте метода и его параметры. Ответ на запрос
также представляет собой JSON.

4. GraphQL. Он поддерживается, например, клиентом Geth. Такой подход позволяет указать, какие поля
объектов вам требуются как часть запроса, снимая потребность в заполнении ненужных полей в ответе. 
Это также позволяет объединять несколько запросов JSON-RPC в один GraphQLзапрос,
благодаря чему снижаются накладные расходы и растёт производительность.

5. Можно, используя web3.eth.accounts управлять аккаунтами (например, создавать методом create) или
используя web web3.eth.accounts, например, получить информацию о провайдерах (providers метод).
Также возможно взаимодействовать с ENS при помощи web3.eth.ens
